About this website.

  • This is a simple website for chatting with all online members.
  • This is a kind of group chatting website.
  • This website is build using HTML CSS JS in frontEnd and NodeJs in backEnd with Socket IO library.
  • By using Socket Io library is able to get all messages without refreshing the page.
  • User can see how many membres are online currently (in top right header section)
  • Anyone can chat with all online persons by anonymously or by telling his/her name.
  • If you just want to test this website and no one is online currently then please open same link in different tab and chat with yourself to test out.
